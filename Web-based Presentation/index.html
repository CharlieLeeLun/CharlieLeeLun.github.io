Introduction

Nowadays, cloud computing software, such as Google Drive, Google Doc and Slide, have been becoming popular among students. One of the advantageous features is that the content created can be accessed anywhere, as long as you remember your user account. Thus, it is needless to say that this is the main reason why cloud computing is so popular. Perhaps another reason is that the software mentioned allow simultaneous edit that previous technology disallowed.
